# WIRE BLOOM Roadmap

今後の実装候補と整理メモです。優先順位はプレイ感と調整しやすさを見ながら更新します。

## Balance

- 敵密度、敵速度、EXP量、Level UP頻度の調整
- 高HP敵と子敵スポーンの出現テンポ調整
- Enemy Densityのリスク/リターン調整
- shockwave連鎖の強さと見やすさの調整
- コンボ維持猶予、ミスペナルティ、報酬感の調整

## Combat / Upgrades

- パルス威力、範囲、角度の追加強化
- HP回復や防御系アップグレードの拡張
- 敵タイプごとの特殊挙動
- ボス固有パターンの追加
- Level UP候補のレアリティやカテゴリ整理

## Visual / Feedback

- ワイヤー接続やブルーム表現の試作
- 敵タイプごとの視認性改善
- EXP吸収、shockwave、コンボ節目の演出強化
- UIレイアウトの継続整理
- フォントアセット検討

## Audio

- 仮SEパラメータの調整
- 連続撃破時の音量/ピッチ制御改善
- BGMまたは環境音の検討
- 将来的な外部音声素材の導入

## Systems

- Game Over以外のラン終了条件
- Records項目の追加
- デバッグ/バランステスト機能の拡張
- セーブデータ形式のバージョン管理
- パフォーマンス確認と必要に応じたオブジェクトプール化

## Distribution

- Electronによるデスクトップアプリ化
- Steam配布を見据えたビルドフロー整備
- ゲームパッド対応の検討
