# WIRE BLOOM i18n Reference

This file is a developer reference for UI localization.
The actual runtime localization data is stored in:
src/config/localization.ts

src/config/localization.ts is the source of truth for in-game text.
This reference file is not loaded by the game.

When adding or changing UI text:
1. Update src/config/localization.ts
2. Update this reference file
3. Check both Japanese and English UI in-game
